The Eventpop API lets you build event management tools, ticket validation apps, and custom user experiences. Access events, tickets, orders, and user profiles using OAuth 2.0 authentication.

## Prerequisites

To use Eventpop API, please ensure that you have:

- **An organizer account** - Your Eventpop organizer account that owns the events you want to integrate with
- **An event** (optional) - While not strictly required, you'll need events to test ticket and order endpoints
- **API Application credentials** - Client ID & Secret obtained by [requesting a new API Application](./eventpop-api-application)
